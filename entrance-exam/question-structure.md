# 入学試験問題構造ガイド / Entrance Exam Question Structure Guide

このファイルは、入学試験問題の標準的な構造と書式設定について説明します。

**フォント指定:**
- 日本語: MS Gothic
- 英語: Century

---

## 問題全体の構造 / Overall Question Structure

### ヘッダー部分 / Header Section
```
**[3]** 以下の対話を読み、次の各問に答えなさい。

**Story Title**
```

### 物語本文 / Story Text Structure

#### 1. 導入部 / Introduction
```
Once upon a time, [setting description]. [Character introduction].
```

#### 2. 対話形式 / Dialogue Format
```
Character Name : Dialogue text here.
Other Character : Response with embedded blank ( ① ).
```

#### 3. ナレーション挿入ボックス / Narrative Insertion Boxes
```
| A | Narrative description for insertion exercises. |

続きの対話...

| B | Another narrative segment. |

さらに続きの対話...

| C | Final narrative segment. |
```

#### 4. 空欄の表記 / Blank Notation
- 選択肢問題用: `( ① )`, `( ② )`, `( ③ )`
- 下線部問題用: `( あ )`, `( い )`, `( う )`

## 問題セクション構造 / Question Section Structure

### **問1.** 語句選択問題 / Vocabulary Selection
```
**問1.** 本文中の空欄①、②に入る語句を適切な表現を以下の選択肢アーエの内からそれぞれ1つずつ選び、記号で答えなさい。

(1) ア option1　　　　イ option2　　　　ウ option3　　　　エ option4
(2) ア option1　　　　イ option2　　　　ウ option3　　　　エ option4
```

### **問2.** ナレーション挿入問題 / Narrative Insertion
```
**問2.** 本文中の空欄| A |〜| C |にナレーションを入れる時、以下の選択肢アーウの内からそれぞれ最も適切な番号を1つずつ選び、記号で答えなさい。

ア Narrative option 1
イ Narrative option 2  
ウ Narrative option 3
```

### **問3.** 理由説明問題 / Reasoning Questions
```
**問3.** 下線部（③）について、[キャラクター名]がこのように言った理由はなんですか。以下の（　）に日本語を埋め、解答を完成させなさい。

[キャラクター名]は（　　1　　）[理由の前半]、（　　2　　）[理由の後半]から。
```

### **問4.** 話し方・様子分析問題 / Speaking Style Analysis
```
**問4.** 下線部（②）について、話者はどのような様子で話していると思いますか。以下の選択肢アーエの内から最も適切な番号を1つ選び、記号で答えなさい。

ア [様子1]
イ [様子2]
ウ [様子3]  
エ [様子4]
```

### **問5.** 内容理解問題 / Content Comprehension
```
**問5.** 以下の選択肢アーエの文について、本文の内容を正しく述べているものはどれか。誤りがあるものについて答えなさい。

ア [内容文1]
イ [内容文2]
ウ [内容文3]
エ [内容文4]
```

## 語彙注釈セクション / Vocabulary Notes Section

```
(注) word1 : 日本語訳1　　word2 : 日本語訳2　　word3 : 日本語訳3
　　 word4 : 日本語訳4　　word5 : 日本語訳5　　word6 : 日本語訳6
```

**注釈の書式規則:**
- `(注)` で開始
- 単語 : 日本語訳の形式
- 2行構成（1行目は(注)から開始、2行目は全角スペース4つでインデント）
- 各項目間は全角スペース2つで区切り

## フォーマット規則詳細 / Detailed Formatting Rules

### 文字表記規則 / Character Notation Rules
- **選択肢**: ア、イ、ウ、エ（カタカナ）
- **空欄番号**: ①、②、③（丸囲み数字）
- **下線部**: （あ）、（い）、（う）（ひらがな、全角括弧）
- **問題番号**: **問1.** 形式（太字、半角ピリオド）

### インデント規則 / Indentation Rules
- 対話の継続行は12文字分インデント
- 語彙注釈の2行目は全角スペース4つでインデント
- ナレーションボックスは独立行として配置

### 太字・強調規則 / Bold/Emphasis Rules
- 問題番号: **問1.**
- セクションタイトル: **[3]**
- 物語タイトル: **Story Title**
- 代名詞強調: **It**（文脈により）

### 空欄・回答欄規則 / Blank/Answer Field Rules
- 選択式空欄: `( ① )`（全角括弧、丸囲み数字）
- 記述回答欄: `_________________________`（アンダースコア25個）
- 日本語記述用: `（　　1　　）`（全角括弧、全角スペース）

## 典型的な物語パターン / Typical Story Patterns

### 1. 動物寓話型 / Animal Fable Type
- 動物キャラクターによる道徳的教訓
- 対比構造（勤勉vs怠惰など）
- 季節の変化を背景とした展開

### 2. 冒険・英雄譚型 / Adventure/Hero Type  
- 主人公の成長と試練
- 救助・助け合いのテーマ
- 困難克服による達成感

### 3. 職人・創作型 / Craftsman/Creation Type
- 技術や芸術への取り組み
- 努力と才能の関係
- 社会的評価と自己実現
