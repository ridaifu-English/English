Pattern 6-4: 同格のthatを使える名詞（3）

3 発言・提案

suggestion（提案）、proposal（提案）、claim（主張）、request（要求）、advice（助言）

The suggestion that we change the plan was well-received.

計画を変更するという提案は好意的に受け取られました。

The claim that the product is completely safe needs to be verified.

その製品が完全に安全だという主張は検証される必要があります。

⚠️ 使えない名詞：具体的な物や人を指す名詞（例：book, car, person など）には同格のthatは使えません
