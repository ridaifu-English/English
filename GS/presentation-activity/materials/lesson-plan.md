| コマ数 | 内容と目標                                                                 | 提出物・やること                                         |
| :----- | :------------------------------------------------------------------------- | :------------------------------------------------------- |
| **1**  | **オリエンテーション＆トピック選定**<br> - 活動全体の流れと評価方法を理解する<br> - 先生から5つのトピック提示、グループ分け<br> - グループで担当トピックを1つ選ぶ<br> - モデルスクリプトを見る（発表のイメージを掴む） | グループで担当トピックを決定し、先生に報告                 |
| **2**  | **調べ学習①：情報収集の開始**<br> - 選んだトピックについて、何を中心に調べるか計画する<br> - インターネットや資料を使って情報収集を開始する<br> - 集めた情報をメモする（ワークシート活用）       | 調べた情報や参考にしたURLなどを記録（ワークシート）        |
| **3**  | **調べ学習②：情報整理と共有**<br> - 引き続き情報収集<br> - グループ内で集めた情報を共有し、スクリプトで伝えたいポイントを話し合う<br> - スクリプトの簡単な構成案（流れ）を考える            | グループ内でスクリプトの骨子を固める                      |
| **4**  | **スクリプト作成①：ドラフト作成**<br> - 調べた情報や話し合った構成案をもとに、英語でスクリプトを書き始める<br> - まずは形にすることを目標に！                            | スクリプトの初稿（ドラフト）を作成する                     |
| **5**  | **スクリプト作成②：推敲と先生チェック**<br> - ドラフトをグループ内で読み返し、より自然で分かりやすい英語表現にする<br> - 調べた情報が効果的に盛り込まれているか確認する<br> - 先生に中間チェックをしてもらう | グループで推敲したスクリプトを先生に見せる                |
| **6**  | **スクリプト完成＆発表準備**<br> - 先生のアドバイスをもとに最終調整し、スクリプトを完成させる！<br> - 発表の役割分担（読む順番）、読み合わせ練習                 | **完成版スクリプト（データまたは紙）**、発表練習             |
| **7**  | **発表会！**<br> - 各グループが作成したスクリプトを発表（1グループ3分厳守）<br>   - **1つのスクリプトを、グループメンバーが分担して順番に読み上げる**<br> - 他のグループの発表を聞いて、内容や伝え方を学ぼう！              | 全グループ発表完了！<br>**最終版スクリプト提出**           |


**評価について（予定）：**

*   **スクリプトの内容:** トピックに関する情報が正確かつ効果的に盛り込まれているか、構成の分かりやすさ、論理的な流れなど
*   **英語表現:** 文法・語彙の正確さ、聞き取りやすさなど
*   **グループへの貢献度:** 協力して調べ学習や作成に取り組めたか
*   **発表の出来栄え:** 聞き取りやすさ、スムーズな交代、調べた内容が伝わる工夫など

**グループワークのポイント：**

*   **協力しよう！** 調べ学習もスクリプト作成も、みんなで力を合わせよう。
*   **役割分担しよう！** 調べる係、まとめる係、英語が得意な人、発表が得意な人など、得意を活かそう。
*   **時間を守ろう！** 各コマの目標を意識して、計画的に進めよう。
*   **正確な情報を伝えよう！** 調べたことは、できるだけ正確に伝えられるように心がけよう。
