# English Grammar Exercise - Part 1

## 1. ( )内から適切な語を選び、英文を完成させてみよう。

**(1)** 劇場には人がほとんどいませんでした。  
There were (little / **few**) people in the theater.

**(2)** 冷蔵庫にはほとんど牛乳はありませんでした。  
There was (**little** / few) milk in the fridge.

**(3)** 彼女の言うことがほとんど聞こえません。  
I can (**hardly** / hard) hear her.

**(4)** すべての生徒がカフェテリアを利用するわけではありません。  
Not (**every** / all) of the students use the cafeteria.

**(5)** そのクラスのすべての生徒が中国語を話すわけではありません。  
(**Not** / None) every student in the class speaks Chinese.

## 2. [ ]内の語句を並べかえて、英文を完成させてみよう。

**(1)** この村の人々のすべてが満足しているというわけではありません。  
[ satisfied / the people / not / this village / all / are / in ]  
**Answer:** Not all the people in this village are satisfied.

**(2)** 去年はその祭に参加した人はほとんどいませんでした。  
[ few / the festival / joined / last / people ] year.  
**Answer:** Few people joined the festival last year.

**(3)** お金は幸福をもたらすというのは、常に真実というわけではありません。  
It [ always / true / not / that / is ] money brings happiness.  
**Answer:** It is not always true that money brings happiness.

**(4)** その2つの物語の間にはほとんど違いがありません。  
[ the two stories / difference / there / between / little / is ].  
**Answer:** There is little difference between the two stories.

**(5)** そのけがをしたネコはほとんど動けませんでした。  
[ cat / hardly / could / the / move / injured ].  
**Answer:** The injured cat could hardly move.
