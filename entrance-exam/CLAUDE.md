## Repository Overview

This is an English entrance exam preparation repository designed for Japanese students. It contains structured exam materials including reading comprehension tests, answer keys, and vocabulary lists for entrance exam preparation.

## Directory Structure & Content Types

### Current Structure
- `entrance-examp-v1.md` - Main entrance exam with "The Ant and the Grasshopper" reading comprehension
- `entrance-examp-v1-answer.md` - Answer sheet with grading guidelines and example responses  
- `entrance-exam-vocabulary-list.csv` - Comprehensive vocabulary list with 2500+ English words for exam preparation
- `previous-questions/` - Archive of past entrance exam materials
  - `images/` - Original JPG images of exam papers (entrance-exam-MMDD.jpg format)
  - `markdown/` - Converted markdown versions of exam images for editing and reuse

### File Organization Principles
- Source images are preserved in `previous-questions/images/` for reference
- Converted markdown files in `previous-questions/markdown/` enable text editing and content reuse
- Main directory contains current/active exam materials
- Naming convention: `entrance-exam-MMDD` (month-day format) for chronological organization

## Common Workflows

### Image to Markdown Conversion
When converting entrance exam images to markdown:
1. Place original JPG files in `previous-questions/images/`
2. Use Read tool to analyze image content
3. Create corresponding `.md` files in `previous-questions/markdown/`
4. Maintain exact formatting including Japanese characters (ア, イ, ウ, エ)
5. Preserve dialogue structure with proper indentation
6. Include vocabulary notes section with Japanese translations

### Content Structure Recognition
Each exam image typically contains:
- **Problem number**: `**[3]**` format indicating question section
- **Story sections**: Narrative text with embedded blanks `( ① )`
- **Dialogue boxes**: `| A |`, `| B |`, `| C |` for narrative insertion
- **Question sections**: `**問1.**` through `**問5.**` with subsections
- **Vocabulary notes**: Bottom section with (注) header and translations

## Content Creation Standards

### Exam Format Structure
Entrance exams follow a structured format based on classical fables/stories:
1. **Reading Passage** - Age-appropriate story with embedded questions
2. **Multiple Choice Questions** - Fill-in-the-blank with 4 options (ア, イ, ウ, エ)
3. **Reference Questions** - Identify what pronouns refer to
4. **Analytical Questions** - Predict outcomes and consequences
5. **Comprehension Questions** - Extract moral lessons and themes
6. **Summary Completion** - Fill in key vocabulary in context summaries

### Answer Sheet Standards
Answer sheets provide:
- Direct answers for objective questions
- Example responses for subjective questions with grading criteria
- Alternative acceptable answers marked with "または" (or)
- Grading guidelines showing required content elements

### Content Formatting Rules
- **Instructions**: Japanese instructions using 「」 quotation marks
- **Story Titles**: Bold formatting with English and Japanese (katakana/hiragana)
- **Question Numbers**: Bold **問X：** format
- **Answer Lines**: Underscores `_________________________` for written responses
- **Multiple Choice**: ア, イ, ウ, エ format (Japanese characters)
- **Dialogue Format**: Character names followed by colons, proper indentation

### Vocabulary Integration
- Vocabulary list contains base forms and variations (e.g., "go", "went", "gone")
- Words selected to support reading comprehension at appropriate levels
- CSV format for easy import into study tools and applications

## Educational Standards

### Reading Comprehension Focus
- Classic fables and moral stories (Aesop's fables, folk tales)
- Embedded comprehension checks within narrative
- Progressive difficulty from literal to inferential questions
- Cultural bridge between English literature and Japanese educational values

### Story Patterns in Archive
Based on `previous-questions/` materials, common story themes include:
- **Animal fables**: Moral lessons through animal characters (Lion/Fox/Deer/Monkey dynamics)
- **Craft and creativity**: Stories about artisans and their work (woodcutter Jack creating art)
- **Adventure and heroism**: Classic hero's journey narratives (Prince Leo rescuing Princess Mimi)
- **Generosity and kindness**: Stories emphasizing helping others and receiving unexpected rewards
- **Consequences of choices**: Characters learning from good/poor decisions

### Language Skills Assessment
- Vocabulary recognition and usage
- Pronoun reference understanding  
- Cause and effect reasoning
- Moral and thematic analysis
- Summary and synthesis skills

### Assessment Design Principles
- Mix of objective (multiple choice) and subjective (written) questions
- Scaffolded difficulty progression within each exam
- Clear grading criteria for subjective responses
- Alternative correct answers acknowledged in answer keys
- Cultural sensitivity in story selection and moral themes
