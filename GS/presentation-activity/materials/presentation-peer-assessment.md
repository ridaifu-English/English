# プレゼンテーション相互評価シート

このシートは、他者のプレゼンテーションを評価し、建設的なフィードバックを提供するためのものです。各項目を5点満点で評価してください。

---

### 1. 便利な表現の正しい使用

この基準は、授業で学習した表現が、発表の中で適切かつ正確に使用されているかを評価します。

| 点数 | レベル | 説明 |
| :--- | :--- | :--- |
| **5** | **優れている** | 多様な表現が正確かつ効果的に使われている。 |
| **4** | **良い** | 多くの表現が正しく使われているが、多様性に欠ける。 |
| **3** | **普通** | 表現の使用は限定的で、時折誤りが見られる。 |
| **2** | **改善が必要** | 表現の使用が少ない、または誤用が目立つ。 |
| **1** | **不十分** | 表現が全く使われていないか、全て不適切である。 |

---

### 2. 明瞭さと一貫性

この基準は、各文が論理的で分かりやすく、全体として一貫性があるかを評価します。

| 点数 | レベル | 説明 |
| :--- | :--- | :--- |
| **5** | **優れている** | 全ての文が明確で、論理的な流れが非常によい。 |
| **4** | **良い** | ほとんどの文は明確だが、一部に分かりにくい箇所がある。 |
| **3** | **普通** | 文の繋がりが部分的に弱く、時折分かりにくい。 |
| **2** | **改善が必要** | 文がしばしば紛らわしく、論理的な繋がりが欠けている。 |
| **1** | **不十分** | 全体的に構成が不明瞭で、内容の理解が困難。 |

---

### 3. 構成（導入、本論、結論）

この基準は、発表が標準的な構成（導入、本論、結論）に沿っているかを評価します。

| 点数 | レベル | 説明 |
| :--- | :--- | :--- |
| **5** | **優れている** | 導入・本論・結論が明確で、全体のバランスが取れている。 |
| **4** | **良い** | 主要な構成要素は揃っているが、一部のバランスが欠けている。 |
| **3** | **普通** | いずれかの構成要素が弱い、または全体の流れがぎこちない。 |
| **2** | **改善が必要** | いずれかの構成要素が欠けているか、著しく不十分である。 |
| **1** | **不十分** | 明確な構成がなく、論理的な進行が見られない。 |

---

### 4. 独自性

この基準は、内容や視点に独自性があり、聞き手の関心を引くかを評価します。

| 点数 | レベル | 説明 |
| :--- | :--- | :--- |
| **5** | **優れている** | テーマや視点が独創的で、非常に興味深い。 |
| **4** | **良い** | 独自の視点や情報が含まれており、関心を引く。 |
| **3** | **普通** | 内容はありきたりで、独自の視点が少ない。 |
| **2** | **改善が必要** | 独自の分析や意見がほとんど見られない。 |
| **1** | **不十分** | 内容に全く独自性がなく、情報の模倣に終始している。 |

---

### 5. チームワーク

この基準は、グループ全員が発表に貢献しているかを評価します。

| 点数 | レベル | 説明 |
| :--- | :--- | :--- |
| **5** | **優れている** | 全員が均等かつ積極的に参加し、連携が取れている。 |
| **4** | **良い** | ほぼ全員が参加しているが、貢献度に若干の差がある。 |
| **3** | **普通** | 一部のメンバーの貢献が明らかに少ない。 |
| **2** | **改善が必要** | 発表が一部のメンバーに偏り、協力体制に課題がある。 |
| **1** | **不十分** | チームとして機能しておらず、個人発表のようになっている。 |
