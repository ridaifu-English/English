# English Grammar Exercise - Part 2

## 3. （　　）内に適切な語を入れて、英文を完成させてみよう。

**(1)** 私はまだこれらの本をすべて読んだわけではありません。  
I ( ) read ( ) of these books yet.  
**Answer:** I haven't read all of these books yet.

**(2)** すべての従業員が勤勉というわけではありません。  
( ) ( ) the employees ( ) hardworking.  
**Answer:** Not all the employees are hardworking.

**(3)** 私は彼の言葉がほとんど信じられませんでした。  
I could ( ) ( ) his words.  
**Answer:** I could hardly believe his words.

**(4)** この町のみんながこの図書館を利用しているわけではありません。  
( ) ( ) in this town uses this library.  
**Answer:** Not everyone in this town uses this library.

**(5)** 科学の進歩がいつも私たちの幸福につながるとは限りません。  
The progress of science ( ) ( ) lead to our happiness.  
**Answer:** The progress of science doesn't always lead to our happiness.

**(6)** 私は9時を過ぎたら甘い物はほとんど食べません。  
I ( ) ( ) sweets after nine.  
**Answer:** I hardly eat sweets after nine.
